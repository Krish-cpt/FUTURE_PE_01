## 🧠 Prompt Engineering Strategy

### 🔹 Approach Used

* **Role-based prompting**
  → Defined AI role as a professional website copywriter

* **Context-based prompting**
  → Included business name, location, and target audience

* **Structured prompting**
  → Clearly defined sections (Homepage, Services, CTA)

* **Tone specification**
  → Modern, stylish, friendly, and professional

---

### 🔹 Prompt Versioning

**Version 1: Basic Prompt**

* Simple instructions
* Generic output

**Version 2: Improved Prompt**

* Added audience targeting
* Added tone and structure
* Result: More engaging and conversion-focused content

---

## 📝 Prompt Used

```
Act as a professional website copywriter.

Create high-converting website content for a local hair salon.

Business Name: Vogue Hair Salon
Location: Sangareddy, Telangana
Address: 4-7-3/4/1/5/A/1, Marepally, Balaji Nagar, Ahmed Nagar

Target Audience:
Men and women aged 18–45 looking for modern hair styling and grooming services.

Generate:
1. Homepage
2. Services Page
3. Call-To-Action section

Tone:
Modern, stylish, friendly, and professional.

Make content simple, persuasive, and website-ready.
```
